1: Para importar o banco de dados basta ir em Server -> Data Import -> Import From Self Contained File e finalmente em Start Import.

2: Para utilizar o sistema basta acessar a o arquivo "config.php" e modificar os dados referentes à seu banco de dados local.

3: Para ter acesso às funcionalidades do sistema, o mesmo deve ser acessado através de algum usuario que esteja disponível no banco de dados, lembrando que há uma distinção entre usuarios normais e administradores, portanto para ter acesso total a todas as funcionalidades do sistema deve-se fazer login com algum usuario que possua o atributo "admin" definido como "1".
